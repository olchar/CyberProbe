🔍 Security Investigation Decision Tree

This decision tree provides a comprehensive workflow for security incident investigation, covering all major incident types:

  • Threat Intelligence: IOC matches with IP/domain/hash enrichment
  • DLP/Data Exfiltration: Data loss prevention incidents with insider risk analysis
  • Compromised Identity: Authentication anomalies, impossible travel, brute force attacks
  • Malware Detection: Malicious file execution with sandbox analysis
  • Insider Risk: High-risk user behavior with enhanced monitoring
  • Network Compromise: Device compromise investigations with forensic collection
