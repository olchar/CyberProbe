🤖 Security Copilot Agent Workflow

This diagram illustrates the Network Device Investigation Agent execution flow in Microsoft Security Copilot:

  • Agent Orchestration: 9-step automated workflow triggered by incident ID
  • Data Source Integration: Microsoft Defender XDR, Sentinel, Threat Intelligence, Log Analytics
  • Skillset Dependencies: Fusion, Generic, NL2KQL, Threat Intelligence DTI
  • Enrichment Layer: Microsoft TI, VirusTotal, and future custom plugin integration
  • Output Generation: Executive HTML reports with MITRE ATT&CK mapping, JSON data, KQL queries

Agent Workflow Steps
1.GetIncident - Retrieve incident details from Defender XDR/Sentinel
2.GetIncidentEntities - Extract entities (IPs, users, devices) from incident
3.ExtractIndicatorsOfCompromise - Identify IOCs from incident data
4.GetSummaryForIndicators - Get threat intelligence summaries
5.GetReputationsForIndicators - Check IOC reputations across sources
6.NL2KQLDefenderSentinel - Generate and execute KQL queries
7.SummarizeData - Consolidate investigation findings
8.AnalyzeSecurityData - Perform correlation and analysis
9.GenerateReportFromTemplate - Create final executive report
